Projeto 1 - Buscador de produtos

Projeto desenvolvido para explorar o consumo de APIs utilizando JavaScript, HTML, CSS e Axios, com o backend rodando em Bun.js. A aplicação permite buscar itens da Amazon por palavra-chave, exibindo os resultados com nome, imagem e preço em um layout responsivo e simples. A API foi construída para intermediar a consulta aos dados, organizando e retornando as informações relevantes ao frontend. O objetivo do projeto foi praticar requisições HTTP, manipulação de JSON, montagem de layouts com foco em usabilidade e responsividade, além de explorar a performance do Bun

Projeto 2 - Forms

Formulário simples para venda de carros desenvolvido com HTML e CSS, focado em aprender a estruturação de formulários, coleta de dados básicos do veículo e do vendedor, com design limpo e organizado para facilitar o preenchimento

Projeto 3 - Blog

Projeto de blog construído com HTML e CSS, voltado para a prática de estrutura semântica e diferentes modelos de layout. O site inclui seções como cabeçalho, área de posts, barra lateral e rodapé, organizados com elementos como header, main, article, aside e footer. O objetivo foi compreender a importância da marcação semântica para acessibilidade e organização do conteúdo, além de experimentar a montagem de diferentes disposições visuais usando apenas HTML e CSS.

Projeto 4 - Tabuada

Aplicação simples desenvolvida com HTML, CSS e JavaScript, com o objetivo de praticar manipulação de DOM e lógica de programação. O usuário insere um número e a aplicação exibe a tabuada correspondente (de 1 a 10) de forma dinâmica na tela. Esse projeto foi uma oportunidade para treinar eventos de formulário, criação de elementos via JavaScript e exibição de resultados de forma clara e organizada, reforçando conceitos fundamentais da linguagem

Projeto 5 - Gerador de QRCODE

Aplicação desenvolvida com HTML, CSS e JavaScript, focada em praticar o consumo de bibliotecas externas e a integração com funcionalidades visuais. O usuário digita um texto ou link, e a aplicação gera um QR Code correspondente instantaneamente na tela. O projeto serviu para explorar a interação entre inputs e saídas gráficas, reforçar a manipulação de DOM e aplicar bibliotecas como qrcode.js para gerar os códigos dinamicamente.

Projeto 6 - Login Instagram

Réplica da página de login do Instagram feita com HTML e CSS, focada em reproduzir fielmente o layout e o estilo visual, para praticar estruturação semântica e design responsivo, sem funcionalidade dinâmica.

Projeto 7 - Single Page

Single page simples criada com HTML e CSS dedicada à acácia, apresentando informações organizadas em seções claras com cabeçalho, conteúdo principal e rodapé. Projeto focado em estruturação básica e estilização visual, sem uso de scripts.

Projeto 8 - Calculadora

Calculadora inspirada no design do iPhone, desenvolvida com HTML, CSS e JavaScript puro. A aplicação reproduz a interface elegante e funcional da calculadora nativa, incluindo botões responsivos, operações básicas e um display dinâmico que atualiza conforme a entrada do usuário

Projeto 9 - SurfPage

Página de surf criada com HTML e CSS, apresentando um visual clean e vibrante que destaca imagens e informações sobre surf. O layout é organizado em seções, com uso de cores e tipografia que remetem à temática praiana e esportiva.

Dashboards - Meteora

Dashboard criado no Excel para uma loja de roupas, onde realizei o tratamento dos dados e construí gráficos para visualizar tendências de vendas e desempenho por categoria, facilitando a análise e tomada de decisões.

Dashboards - Cafeteria

Dashboard avançado desenvolvido no Excel para uma cafeteria, envolvendo o tratamento e análise de grandes volumes de dados. O projeto foca em indicadores principais como despesas, lucros e produtos mais vendidos, apresentando gráficos detalhados e tabelas dinâmicas que facilitam o monitoramento financeiro e estratégico do negócio
